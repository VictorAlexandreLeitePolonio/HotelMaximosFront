# HotelMaximosFront

Frontend do sistema Hotel Maximos.

## Stack

- React 19.2.1
- TanStack Start
- TanStack Router
- TanStack Query
- TanStack Table
- Zustand
- Zod
- shadcn/ui
- Motion
- Cypress

## Setup local

```bash
npm install
npm run dev
```

Aplicação local:

```txt
http://localhost:3000
```

## Scripts

```bash
npm run dev
npm run build
npm run typecheck
npm run test:e2e
```
